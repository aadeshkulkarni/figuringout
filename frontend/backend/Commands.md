### Prisma Commands sheet

- Define schema in prisma/schema.prisma file    
- `npx prisma migrate dev --name init_schema`
- `npx prisma generate --no-engine`
- `npm install @prisma/extension-accelerate`    


### Backend deployment
- `npx wrangler login`
- `npx wrangler whoami`
- `npm run deploy`
